# 🚗 AutoCard Editor Pro - Stellantis / Nissan

Editor automatizado de tarjetas (Cards) publicitarias para el sector automotriz. Este script permite actualizar precios, promociones, beneficios e imágenes de Cloudinary de forma masiva sin editar HTML manualmente.

## 🚀 Funcionalidades
- **Memoria de Sesión:** Edita múltiples vehículos y descarga el catálogo completo en un solo paso.
- **Formateo Inteligente:** Detecta automáticamente símbolos de `$` y `%` para aplicar estilos resaltados.
- **Gestión de Cloudinary:** Genera automáticamente `srcset` optimizados para carga rápida en Google Sites.
- **Inventario Especial:** Activación dinámica de promociones de unidades disponibles (descomenta HTML).
- **Añadir Unidades:** Botón dedicado para insertar nuevas tarjetas al catálogo existente.

## 🛠️ Tecnologías
- **Python 3.10+**
- **Streamlit:** Interfaz de usuario web.
- **BeautifulSoup4:** Manipulación y parsing de DOM HTML.
- **Re (Regex):** Lógica de búsqueda y reemplazo de patrones monetarios.

# AutoCard Promotion Editor

## Goal
A specialized tool designed to standardize and automate the creation of digital promotion cards for automotive catalogs (Jeep, RAM, Fiat).

## Business Impact
* **Consistency:** Ensured 100% brand alignment across all digital platforms.
* **Speed:** Faster update cycles for seasonal promotions and pricing changes.
* **Quality Control:** Automated the generation of HTML/CSS cards to prevent manual formatting errors.

## Tech Stack
* **HTML/CSS:** Custom card layouts for digital catalogs.
* **Content Engineering:** Automated template filling for dynamic pricing.

## Key Features
* Interactive card editing for different vehicle brands.
* Responsive design for mobile and desktop viewing.
* Direct integration with digital sales platforms.
